1. اسکریپت ها بهمون این قابلیت رو میدن که ما بتونیم یسری کامند هاکه توی تی پروژه به اونا به تعداد زیادی نیاز داریم رو توی این قسمت داخل کلید هایی ذخیرشون بکنیم و بجای نوشتن خود اونم کد 
    توی قسمت اسکریپت پکیج دات جیسون میتونیم اسکریپت هایی رو بنویسیم که هر موقع میخایم اون اسکریپت اجرا بشه بجای این همه خط کد توی ولیو اسکریپت فقط بیایم بگیم
    npm run test 
        اگه ما این اسکریپت رو احرا کنیم برای ما میاد این متن و کد رو اجرا میکنه بجای خط کد بالا ما میتونم حتی بنویسیم
            npm run-script [script name]
                ان پی ام اجرا کن یکدونه اسکریپت به اسمه ... 

2. ما اگر بخایم به کمک نود جی اس یک فایل جاوا اسکریپت رو اجرا بکنیم توی این قسمت میگیم که 
    node [js file name]
        ما اگر نخایم هر دفعه نیایم بنویسیم نود اسم فایل بیایم داخل ابجکت اسکریپت یک کی دیگه تعریف میکنیم با اسم اختیاری مثلا ایندکس میایم ولیش عملکردی که میخایم بعد از اجرای اون خاست رو مینویسیم
            "[my coustom name]": "node [package name]"

3. tsc [type script file name].ts
    برای ترجمه فایل تایپ اسکریپت به جاوا اسکریپت من نمیخایم بیام هی کد بالا رو بزنم پس من از کد زی استفاده میکنم
        tsc [type script file name].ts --watch / tsc [type script file name].ts -w
            ما وقتی براش فلگ واچ میزاریم میاد روی اسم فایل دات تایپ اسکریپت واچ انجام میده
                یعنی روش نظارت میکنه و به محض سیو فایل تغییرات ترجمه میشه 
                    در قسمت tsc پکیج جیسون اگر بنویسیم 
                        "app": "tsc [type script file name].ts -w"
                            بازم کد های بالا رو برام میاره و فرقی نداره مثل وریبل در سی اس اس هست 

4. برای متوقف کردن یک کامند میتونم از شورت کات زیر استفاده کنیم
    CTRL + C
