1. وقتی میایم مینویسیم ان پی ام ای بوت استرپ به عنوان مثال میره اخرین نسخه میجورشو نصب میکنه 

2. برای نصب  نسخه دلخواه از یک پکیجی کد زیر رو مینویسیم 
    npm i [package name]@[coustom version]
    npm i bootstrap@4 : 
        در این حالت میره اخرین نسخه مینور و پچ رو نصب میکنه و نسخه چهار رو ثابت نگه میداره
    npm i bootstrap@4.2 :
        در این نسخه میجور و مینور رو ثابت نگه میداره و فقط اخرین نسخه پچ رو نصب میکنه
    npm i bootstrap@4.2.1 :
        فقط همین نسخه رو نصب میکنه
