1. command  چیه
کامند یک خط کدی هست که به ترمینال میدیم و واسمون یک کاری رو انجام میده

2. cd چیه
برای وارد شدن به مسیری هست

3. cd مخفف چیه ؟ 
change directory

4. برای برگشت از یک پوشه باید چه کامندی استفاده کنیم ؟ 
cd ../ برای برگشت یک پوشه به عقب

5. ما میتونیم در گیت کد هایی که قبلا نوشتیم رو بدون نوشتن دوباره بیاریم با دکمه های فلش رو به بالا

6. برای ساخت فولدر در گیت میام مینویسم 
mkdir react

7. mkdir اصلا یعنی چی ؟
یعنی بیا برای من یکدونه دایرکتوری بساز و سپس اسپیس اسم فولدری که میخایم ساخته بشه

8. برای حذف و تمیز کردن گیت میایم از کامند کلیر استفاده میکنیم
clear

9. برای حذف یک پوشه در گیت میایم مینویسیم
rmdir => remove directory

10. اگر مسیر پروژه خیلی طولانی هیت میتوانیم بریم داخل پوشه پروژه و کلیک راست کنیم و سپس گزینه زیر رو بزنیم
git bash here

11. ls چیه ؟ 
ال اس به معنای لیست هست یعنی وقتی توی دایرکتوری هستیم و میخام لیست فایل و فولدر هایی که توی اون دایرکتوری هست رو ببنیم میایم از کامند ال اس استفاده میکنیم
